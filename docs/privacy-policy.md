## 한국어

월급이(WorkPay)(이하 "본 앱")는 **사용자의 어떠한 개인정보도 수집하지 않습니다.**

1. **수집하는 정보**: 없음. 본 앱은 회원가입·로그인·계정 기능이 없습니다.
2. **데이터 저장**: 사용자가 입력한 근무처, 근무 일정, 앱 설정 등 모든 데이터는 **사용자 기기 내부에만** 저장되며 외부로 전송되지 않습니다.
3. **네트워크·추적**: 본 앱은 사용자의 데이터를 외부 서버로 전송하지 않습니다. **광고, 분석(애널리틱스), 추적 도구를 일절 사용하지 않습니다.**
4. **백업**: 백업 파일은 사용자가 직접 "내보내기"를 실행할 때만 생성되며, 사용자가 직접 선택한 위치 또는 공유 대상에만 저장됩니다. 개발자는 이 파일에 접근할 수 없습니다.
5. **제3자 제공**: 없음.
6. **아동**: 본 앱은 개인정보를 수집하지 않으므로 모든 연령대가 안전하게 사용할 수 있습니다.
7. **권한**: 백업 내보내기/가져오기를 위해 파일 선택·공유 기능을 사용할 수 있으나, 이는 사용자가 직접 실행한 경우에만 동작합니다.

## English

WorkPay ("the App") **does not collect any personal data.**

1. **Data collected**: None. The App has no account, login, or sign-up.
2. **Storage**: All data you enter (jobs, work shifts, app settings) is stored **only on your device** and is never transmitted elsewhere.
3. **Network & tracking**: The App does not send your data to any server. It uses **no ads, no analytics, and no tracking** of any kind.
4. **Backups**: Backup files are created only when you explicitly export them, and are saved only to a location or share target that you choose. The developer cannot access these files.
5. **Third-party sharing**: None.
6. **Children**: The App is safe for all ages because no data is collected.
7. **Permissions**: File picker / share features may be used for backup export/import, and only operate when you explicitly trigger them.
