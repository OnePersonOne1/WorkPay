# Google Play 제출 체크리스트 (월급이 / WorkPay)

AAB: `build/app/outputs/bundle/release/app-release.aab` (release 업로드 키 서명됨)
앱 아이콘(512): `assets/icon/play_store_icon_512.png`

---

## 1. 데이터 보안 (Data safety) 양식 답변
Play Console > 앱 콘텐츠 > 데이터 보안

- **앱이 데이터를 수집하거나 공유합니까?** → **아니요 (No)**
  - 이 앱은 사용자 데이터를 수집/전송하지 않음. 모든 데이터는 기기 로컬에만 저장.
- **데이터가 전송 중 암호화됩니까?** → 해당 없음(데이터 전송 자체가 없음)
- **사용자가 데이터 삭제를 요청할 수 있습니까?** → 앱 데이터는 기기에만 있으므로 앱 삭제 시 함께 삭제됨
- **독립적 보안 검토(선택)** → 아니요

> 핵심: 모든 항목 "수집 안 함 / 공유 안 함". 이 앱엔 매우 유리합니다.

---

## 2. 콘텐츠 등급 설문 (Content rating)
Play Console > 앱 콘텐츠 > 콘텐츠 등급

- **카테고리**: 유틸리티 / 생산성 / 도구 (Utility, Productivity)
- 폭력·성적 콘텐츠·욕설·도박·약물 등 모든 항목 → **아니요**
- 사용자 간 소통/공유 기능 → **아니요** (오프라인, 계정 없음)
- 위치 공유 → 아니요
- 결과 예상 등급: **전체이용가 (Everyone / 3+)**

---

## 3. 기타 앱 콘텐츠 선언
- **광고 포함?** → **아니요** (광고 없음)
- **타깃 연령층**: 만 18세 이상 또는 전체 — 학생/성인 대상 유틸리티이므로 13+ 또는 전체 선택 가능
  (아동 대상 앱 아님 → "어린이용 앱" 프로그램 대상 아님으로 선택)
- **정부 앱?** → 아니요
- **뉴스 앱?** → 아니요
- **코로나19 앱?** → 아니요
- **데이터 보안**: 위 1번 참조

## 4. 그래픽 자산 (Play Console 업로드용)
| 자산 | 규격 | 상태 |
|---|---|---|
| 앱 아이콘 | 512×512 PNG (32-bit) | ✅ `assets/icon/play_store_icon_512.png` |
| 피처 그래픽 | 1024×500 PNG/JPG | ⏳ 필요 — 제작 안내 아래 |
| 휴대전화 스크린샷 | 2~8장, 16:9 또는 9:16, 최소 320px | ⏳ 앱 실행 캡처 필요 |
| 7인치/10인치 태블릿 (선택) | 권장 | 선택 |

### 피처 그래픽 (1024×500)
가로 배너. 앱 아이콘 + "월급이 / WorkPay" + 한 줄 카피
("광고 없는 알바 근무표·월급 계산기")를 흰 배경/브랜드 컬러로 배치.
→ Gemini 프롬프트 또는 간단 디자인으로 만들 수 있음(요청 시 도와드림).

### 스크린샷 (권장 4~6장)
1. 일정표(캘린더) — 근무 입력된 모습
2. 월급 합계 / 명세 화면
3. 근무처 등록/편집
4. 주급·일급 표시 토글
5. 설정(테마·통화 단위·언어)
6. 백업/복원
→ `flutter run -d windows`(또는 Android) 실행 후 캡처. 원하면 제가 앱을 띄워 캡처를 떠드릴 수 있음.

## 5. 개인정보처리방침
- 파일: `docs/privacy-policy.md`
- **공개 URL 필요**. 호스팅 옵션:

  - 또는 GitHub에서 파일 열고 "Raw"/렌더링된 주소
  - 또는 Google Sites / Notion 공개 페이지
- 파일 안의 `[CONTACT_EMAIL]`을 실제 연락 이메일로 교체할 것.

## 6. 출시 트랙 순서 (신규 개인 계정)
1. **비공개 테스트(Closed testing)** 트랙 생성 → AAB 업로드
2. 테스터 **12명 이상** 이메일 등록 + 옵트인
3. **14일간** 비공개 테스트 유지
4. 이후 **프로덕션 출시 액세스 신청** → 검토 통과 시 정식 출시 가능

> 이미 만든 지 오래된/검증된 계정이면 이 단계가 면제될 수 있으니 콘솔 안내를 확인하세요.
